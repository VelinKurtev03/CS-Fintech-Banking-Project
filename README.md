# CS-Fintech-Banking-Project
CS Project for Fintech &amp; Banking


Requirements:

* Категория 2: Fintech & Banking (Висок финансов риск)
* Заявка за кредит: Валидация на доходи и кредитна история.
* Крипто транзакция: Валидация на портфейл (Hash) и сума.
* Искане за овърдрафт: Проверка на лимити в реално време.
* Регистрация на корпоративна карта: Нива на оторизация.
* Валутен обмен: Бързина на транзакцията и актуални курсове (Integrity).
* Възстановяване на парола: Многофакторна автентикация (MFA).
